# 3. การตั้งค่า Configuration

ขั้นตอนสำคัญเพื่อให้หุ่นยนต์ขยับได้ คือการตั้งค่า IP และ Serial Number ให้ถูกต้อง

> **คำเตือน:** หากตั้งค่า IP ผิด หุ่นยนต์จะไม่สามารถเชื่อมต่อกับ LiDAR ได้

## 3.1 แก้ไข Config ของ LiDAR
ไฟล์นี้กำหนดค่า Network ระหว่างบอร์ดกับตัวเซนเซอร์

**คำสั่งแก้ไขไฟล์:**
```bash filename="Bash"
vim src/livox_ros_driver2/config/MID360_config.json
```

>สิ่งที่ต้องตรวจสอบและแก้ไข: ต้องแก้ IP ให้ตรง :

Host IP (บอร์ด): 10.192.1.3

LiDAR IP (เซนเซอร์): 10.192.1.5

>ตัวอย่างโค้ดในไฟล์:

```json filename="JSON" showLineNumbers {9,15}
JSON

{
  "lidar_summary_info": {
    "lidar_type": 8
  },
  "MID360": {
    "host_net_info": {
      "cmd_data_ip": "10.192.1.3",
      "cmd_data_port": 56101
    }
  },
  "lidar_configs": [
    {
      "ip": "10.192.1.5",
      "pcl_data_type": 1
    }
  ]
}
```
